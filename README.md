# daily

> 我的个人日记本，日记文件统一存放在 [`日记/`](Obsidian%20Vault/日记/) 目录下。

## 概述

这是一个以日记为核心的 Obsidian 笔记仓库。日记按日期命名（如 `2026-06-25-00-1.md`），每条独立成篇，记录日常生活中的琐事、随想和感悟。日期即文件名，翻阅目录即可按时间线回溯过往，无需额外索引。

除了日记之外，仓库中也积累了一些学习笔记与生活攻略。比如数学分析中极限理论的系统梳理（ε-δ 语言、数列极限与函数极限的定义及性质定理），以及日常生活中打车省钱的实用技巧。这些内容虽然题材各异，但都是个人知识体系的有机组成部分。

笔记借助 Obsidian 的标签系统和双向链接相互关联，原本散落的记录会随着时间推移逐渐编织成一张个人知识网络。某个日记里提到的想法，可能日后衍生为一篇专题笔记；一篇学习笔记中的概念，也可能在另一篇日记中被引用和反思。这正是 Obsidian 作为知识管理工具的核心价值。

仓库内容会持续更新，日记是主体，其余笔记随兴所至、随需所记。不求面面俱到，但求真实记录，为未来的自己留一份可检索的记忆存档。

## 目录结构

- `Obsidian Vault/日记/` — **日记文件**，按日期归档，仓库的主要内容
- `Obsidian Vault/assets/` — 图片等附件
- `Obsidian Vault/Tags/` — 标签文件
- `Obsidian Vault/.obsidian/` — Obsidian 配置与插件

## 使用

使用 [Obsidian](https://obsidian.md) 打开 `Obsidian Vault` 目录即可。
