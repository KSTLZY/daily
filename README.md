# daily

> 我的日记本。日记文件统一存放在 [`日记/`](Obsidian%20Vault/日记/) 目录下，按日期命名（如 `2026-06-25-00-1.md`），方便检索和回溯。

日常记录以日记为主，记录生活琐事、随想感悟。除此之外也包含一些学习笔记和生活攻略，例如数学分析中的极限理论梳理、打车省钱技巧等。笔记借助 Obsidian 的标签和双向链接相互关联，随时间沉淀成个人知识网络。

## 目录结构

- `Obsidian Vault/日记/` — **日记文件**，按日期归档
- `Obsidian Vault/assets/` — 图片等附件
- `Obsidian Vault/Tags/` — 标签文件
- `Obsidian Vault/.obsidian/` — Obsidian 配置与插件

## 使用

使用 [Obsidian](https://obsidian.md) 打开 `Obsidian Vault` 目录即可。
