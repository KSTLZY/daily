# 相机知识大全

## 一、曝光三要素

### 1.1 光圈（Aperture）

光圈是镜头内可调节的开口，用 **f/ 值**表示。f/ 值 = 焦距 ÷ 通光孔径。

| f/ 值 | 光圈大小 | 进光量 | 景深 |
|-------|----------|--------|------|
| f/1.4 | 极大 | 极多 | 极浅 |
| f/2.0 | 大 | 多 | 浅 |
| f/2.8 | 中大 | 较多 | 较浅 |
| f/4.0 | 中 | 中等 | 中等 |
| f/5.6 | 中小 | 较少 | 较深 |
| f/8.0 | 小 | 少 | 深 |
| f/11 | 很小 | 很少 | 很深 |
| f/16 | 极小 | 极少 | 极深 |
| f/22 | 针孔 | 微光 | 全景深 |

**完整光圈级数（每级相差一倍进光量）**：
f/1.0 → f/1.4 → f/2.0 → f/2.8 → f/4.0 → f/5.6 → f/8.0 → f/11 → f/16 → f/22 → f/32

记忆口诀：每隔一档乘以 √2 ≈ 1.4。

**光圈叶片数**决定焦外散景形状，叶片越多散景越圆润（常见 7-11 片）。

**衍射极限**：光圈太小（如 f/16 以上）时光线绕过叶片边缘产生衍射，画质反而下降。大多数镜头在 f/5.6 ~ f/8 达到最佳锐度。

### 1.2 快门速度（Shutter Speed）

单位：秒 / 分之一秒。

| 速度 | 适用场景 |
|------|----------|
| 1/8000 ~ 1/2000 | 冻结高速运动（飞鸟、运动员） |
| 1/1000 ~ 1/500 | 日常运动（跑步、玩耍的儿童） |
| 1/250 ~ 1/125 | 日常手持，轻度动作 |
| 1/60 ~ 1/30 | 静物手持下限（需防抖辅助） |
| 1/15 ~ 1/4 | 需要稳定支撑，可拍流水拉丝 |
| 1/2 ~ 30s | 长曝光（车轨、星轨、瀑布雾化） |
| BULB | 手动控制时长，超出 30 秒时使用 |

**安全快门法则**：手持拍摄时，快门速度 ≥ 1/焦距（全画幅等效）。如 200mm 镜头至少 1/200s。防抖可下探 2-4 档。

**快门类型**：
- **机械快门**：物理帘幕，有闪光同步速度限制（通常 1/200s 或 1/250s）
- **电子前帘**：减少机震，但可能影响焦外
- **电子快门（全域）**：完全静音、无震动，但可能有果冻效应（读出速度慢时）
- **电子快门（卷帘）**：逐行读取，快速移动会产生倾斜变形

**闪光同步速度**：机械快门帘幕运动决定了使用闪光灯的最高快门速度，通常 1/200s ~ 1/250s。超出此速度需要高速同步（HSS）。

### 1.3 ISO（感光度）

信号增益，非物理进光量变化。

| ISO 范围 | 画质描述 |
|----------|----------|
| 50 ~ 100（基础 ISO） | 最佳动态范围、最少噪点 |
| 200 ~ 800 | 画质轻微下降，日常可用 |
| 1600 ~ 6400 | 噪点明显但可控，暗光常用 |
| 12800 ~ 51200 | 噪点严重，细节损失明显 |
| 51200+ | 扩展 ISO，应急使用 |

**双原生 ISO（Dual Native ISO）**：部分摄影机/相机有两套增益电路，在不同的高 ISO 区间画质不降反升。如索尼 A7S3 的第二档在 ISO 12800 突然变干净。

**ISO 不变性（ISO Invariance）**：某些相机传感器在较低 ISO 欠曝后期补亮的噪点，与直接使用高 ISO 几乎相同。这意味着可以故意欠曝保护高光，后期提亮暗部。

### 1.4 曝光补偿（Exposure Compensation）

在半自动档（A/S/P）下，通过 ±EV 偏移相机测光结果：
- **+EV**：画面变亮（拍雪景、白背景）
- **-EV**：画面变暗（拍黑色主体、夜景）

范围通常 ±3EV 到 ±5EV，步长 1/3 EV。

### 1.5 EV（Exposure Value）与 Stop

- **1 EV = 1 档曝光 = 进光量翻倍或减半**
- 光圈开一档、快门慢一档、ISO 高一档，三者等价互换
- 阳光十六法则：晴天正午，f/16、1/ISO 秒的曝光组合基本正确

## 二、测光模式（Metering Modes）

相机测量场景亮度来决定曝光参数。

| 模式 | 原理 | 适用场景 |
|------|------|----------|
| **评价测光 / 矩阵测光** | 将画面分区，综合计算权重 | 大多数日常场景 |
| **中央重点平均** | 优先画面中央区域（约 60-70% 权重） | 人像、主体居中的场景 |
| **点测光** | 仅测量取景器中心约 1%-5% 的区域 | 高对比场景（舞台、逆光人像） |
| **局部测光** | 介于中央重点和点测光之间，覆盖约 8-15% | 主体占比中等的逆光场景 |
| **高光优先测光** | 优先保护高光不过曝 | 舞台、婚礼（白色婚纱） |

**测光参考基准**：相机默认将场景反射率假定为 **18% 中性灰**。这解释了为什么白雪会被拍成灰雪——相机认为雪"太亮"而降曝光。此时需要 +1.5~+2EV 补偿。

**测光范围（测光表 EV 范围）**：相机测光系统能工作的亮度范围，如 EV 0 ~ EV 20（ISO 100）。超出此范围需手动估算曝光。

## 三、直方图（Histogram）

直方图是亮度分布的柱状图，横轴左黑右白（0-255），纵轴像素数量。

### 读图方法

```
暗部 ← → 亮部
|████  ▂▂▂▂  ▆▆▆▆  ▃▃  |
0                        255
```

- **左墙（贴死左边缘）**：暗部死黑丢失细节
- **右墙（贴死右边缘）**：高光过曝丢失细节（高光溢出）
- **山形居中**：中调为主，曝光正常
- **U 形分布**：高反差场景，暗亮两极多中间少
- **梳子状**：后期暴力调整留下的断层

### ETTR（向右曝光）

在 Raw 拍摄时故意让直方图尽量偏右（但不溢出），后期压暗。原理：
- Raw 的 14bit 线性数据中，每高 1EV 携带的数据量翻倍
- 向右曝光可以最大化暗部细节和信噪比

## 四、焦距与视角

### 4.1 分类

| 类型 | 全画幅焦距 | 视角范围 |
|------|-----------|----------|
| 超广角 | < 24mm | > 84° |
| 广角 | 24mm ~ 35mm | 84° ~ 63° |
| 标准 | 35mm ~ 70mm | 63° ~ 34° |
| 中长焦 | 70mm ~ 135mm | 34° ~ 18° |
| 长焦 | 135mm ~ 300mm | 18° ~ 8° |
| 超长焦 | > 300mm | < 8° |

### 4.2 各焦段特点

**超广角（14-24mm）**
- 极宽视角，前景被放大、远景被推远
- 强调纵深感、引导线
- 风光、建筑、星空必备
- 边缘拉伸失真，人像边缘会变形

**24mm / 28mm**
- 经典风光焦段、街拍
- 手机主摄大约等效 24-28mm
- 室内窄空间也能施展

**35mm**
- "人文之眼"，最接近单眼注意力范围
- 纪实、街拍、环境人像
- 既有人物又有环境叙事

**50mm**
- "标头"，与人眼透视最接近（非视角）
- 无畸变，最自然
- 万能焦段，从人像到静物都能驾驭

**85mm**
- "人像镜皇"
- 透视扁平化适中，不压缩五官也不拉伸
- 工作距离舒适（全身约 5-7m）
- 背景虚化能力突出

**135mm**
- 人像长焦、舞台、体育
- 强烈的空间压缩感
- 主体与背景分离度高

**200mm+**
- 野生动物、体育远摄、月亮太阳
- 空气透视（远处的薄雾感）开始影响画质
- 需要高速快门和高稳定性支撑

### 4.3 视角与距离的关系

**透视不由焦距决定，由拍摄距离决定。** 焦距决定的是取景范围。

- 同一距离用不同焦距拍摄 → 透视相同，画面是长焦画面的裁剪
- 同一主体大小用不同焦距拍摄 → 需要改变距离，透视随之改变

广角贴近人脸 = 鼻子大耳朵小。长焦退远拍人脸 = 五官压平。

## 五、传感器画幅

### 5.1 各画幅参数

| 画幅 | 尺寸 (mm) | 裁切系数 | 代表机型 |
|------|-----------|----------|----------|
| 1/2.3" 手机 | 6.17 × 4.55 | ~5.6× | 旧手机 |
| 1" 型 | 13.2 × 8.8 | 2.7× | Sony RX100 系列、DJI 无人机 |
| M4/3 | 17.3 × 13.0 | 2.0× | OM System、松下 G 系列 |
| APS-C（佳能） | 22.3 × 14.9 | 1.6× | EOS R7/R10/R50 |
| APS-C（其他） | 23.6 × 15.6 | 1.5× | 富士 X 系列、索尼 A6xxx、尼康 Z DX |
| APS-H | 27.9 × 18.6 | 1.3× | 佳能旧 1D 系列（已停产） |
| 全画幅（135） | 36 × 24 | 1.0× | 主流专业/准专业机身 |
| 中画幅（4433） | 44 × 33 | 0.79× | 富士 GFX、哈苏 X 系列 |
| 中画幅（645） | 56 × 41.5 | 0.62× | 飞思 IQ4、哈苏 H 系列 |
| 大画幅 | 4×5 英寸起 | — | 技术相机、胶片 |

### 5.2 等效换算

- **等效焦距** = 实际焦距 × 裁切系数
  - 例：APS-C 机身上 35mm 镜头 ≈ 全画幅 52.5mm 视角
- **等效光圈（景深）** = 实际光圈 × 裁切系数
  - 例：APS-C f/2.0 ≈ 全画幅 f/3.0 的景深效果
- **等效光圈（等效进光量）**：不影响曝光，但影响整体信噪比
  - 更大画幅能收集更多总光量，同等技术下信噪比更高

### 5.3 画幅对画质的影响

- **高感信噪比**：画幅每大一档，同等技术下高感约好 1 档
- **动态范围**：大画幅有更大的满阱容量（Full Well Capacity），高光保留能力更强
- **虚化能力**：同等构图和光圈下，大画幅景深更浅（因为需要更长焦距或更近距离）
- **衍射极限**：大画幅在更小光圈才出现衍射，风光摄影更友好

## 六、对焦系统

### 6.1 对焦方式

| 模式 | 全称 | 行为 |
|------|------|------|
| AF-S | 单次自动对焦 | 半按锁定，适合静止 |
| AF-C | 连续自动对焦 | 持续追踪，适合运动 |
| AF-A | 自动切换 | 相机判断静止/运动自动切换（入门机常见） |
| DMF | 全时手动对焦 | 自动对焦完成后可手动微调，无需切换模式 |
| MF | 手动对焦 | 纯手动，依赖取景器放大、峰值对焦辅助 |

### 6.2 对焦区域模式

| 模式 | 行为 |
|------|------|
| **单点** | 手动选择一个对焦点，精度最高 |
| **区域** | 在选定的区域内自动找主体 |
| **广域 / 全域** | 全屏自动选择，适合无脑拍摄但容易对错 |
| **扩展灵活点** | 主焦点 + 周围辅助点，适合轻微运动 |
| **追踪** | 锁定主体后全屏跟踪，配合 AF-C 使用 |
| **眼控 / 眼部 AF** | 自动检测并跟踪人眼或动物眼 |
| **鸟眼 / 车辆 AF** | 各品牌最新 AI 对焦，识别特定对象 |

### 6.3 对焦点类型

- **对比度检测（CDAF）**：来回推拉找反差最大处，慢但准
- **相位检测（PDAF）**：一次判断方向和距离，快但可能跑焦
- **混合对焦**：PDAF 粗定位 + CDAF 精调，主流无反标配
- **十字对焦点**：对竖线和横线都敏感（比一字型更可靠）

### 6.4 高级对焦技术

**后键对焦（Back-Button Focus）**：
- 将对焦功能从快门按钮分离到 AF-ON 键
- 按下 AF-ON 对焦，松开停止对焦
- 快门只负责释放，不再触发对焦
- 适用场景：运动追焦、陷阱对焦、需要灵活切换 AF-S/AF-C

**陷阱对焦**：
- MF 模式下预先对焦到某个位置，主体到达该面时触发快门
- 拍摄起跑线、鸟巢出入口极有效

**焦点堆叠（Focus Stacking）**：
- 拍摄多张不同焦点位置的照片，后期合成全景深
- 微距、风光近前景远背景时使用
- 部分相机内置自动焦点包围功能

**焦点映射（Focus Mapping / Hyperfocal）**：对焦在超焦距位置，使从前景到无穷远都在焦内。常用于风光。

### 6.5 跑焦与校准

- **AF 微调（AF Fine Tune）**：DSLR 时代镜头和机身可能出现对焦偏移，需要校准（±20 范围）。无反不存在此问题。
- **焦点偏移（Focus Shift）**：某些镜头在改变光圈时焦点位置会移动。大光圈对焦后收光圈拍摄可能导致焦点前移。

## 七、景深（Depth of Field）

### 7.1 影响景深的四个因素

| 因素 | 变化方向 | 景深变化 |
|------|----------|----------|
| 光圈 | f/ 值越大（光圈越小） | 越深 |
| 焦距 | 越短 | 越深 |
| 拍摄距离 | 越远 | 越深 |
| 传感器画幅 | 越小 | 越深（因为同等构图需要更短焦距或更远距离） |

### 7.2 景深分布

对焦平面前后的景深不是对称分布的：
- 焦**前**占约 1/3
- 焦**后**占约 2/3

近距离微距时趋近 1:1。

### 7.3 散景（Bokeh）

焦外部分的成像质量：
- **好散景**：柔和、奶油般过渡、无硬边（Sonnar、大光圈定焦）
- **差散景**：二线性明显（光斑有硬环）、洋葱圈（非球面镜片模具痕迹）、口径蚀（画面边缘光斑呈猫眼形）

**散景形状** = 光圈叶片形状 + 口径蚀影响。圆形光斑在画面中央，边缘变成柠檬形或猫眼形。

## 八、镜头种类与光学

### 8.1 按焦段分类

**定焦镜头（Prime）**
- 优点：画质更好、光圈更大、体积更小、结构简单更可靠
- 缺点：构图靠走，不灵活
- 常见焦段：24mm / 28mm / 35mm / 40mm / 50mm / 85mm / 105mm / 135mm / 200mm / 300mm / 400mm / 600mm

**变焦镜头（Zoom）**
- 优点：灵活构图、一镜多用
- 缺点：画质妥协、最大光圈较小、体积更大
- 常见组合：
  - 大三元：16-35mm f/2.8、24-70mm f/2.8、70-200mm f/2.8
  - 小三元：同焦段但光圈 f/4
  - 天涯镜：24-200mm、28-300mm 超大倍率

### 8.2 按特殊功能分类

**微距镜头（Macro）**
- 1:1 放大倍率（实物在传感器上成像等大）
- 最近对焦距离极近，结构扁平
- 焦段：50mm（短微距）、100mm（中微距/百微）、180mm（长微距，拍昆虫不易惊扰）

**移轴镜头（Tilt-Shift）**
- Tilt（倾斜）：改变焦平面方向 → 创造微缩模型效果或拓宽景深（沙姆定律）
- Shift（偏移）：矫正透视变形 → 建筑摄影保持竖直线不汇聚

**鱼眼镜头（Fisheye）**
- 故意保留桶形畸变，视角 180°
- 对角线鱼眼（~180° 对角）vs 全周鱼眼（圆形像场）

**柔焦镜头（Soft Focus）**
- 故意保留球差，高光周围蒙一层柔光
- 人像复古风格，可调柔化程度

**折返镜头（Catadioptric / Mirror Lens）**
- 使用反射镜缩短物理长度
- 极小体积实现 500mm+ 焦段
- 焦外光斑呈甜甜圈形
- 光圈固定（通常 f/8 或 f/11），手动对焦

### 8.3 按光学结构分类

**标准结构**：
- **Tessar（天塞）**：4 片 3 组，锐度高反差好，经典标头结构
- **Planar（普拉纳）**：6 片 4 组对称双高斯，大光圈标头常用
- **Sonnar（松纳）**：不对称结构，紧凑长焦，焦外优雅
- **Distagon（迪斯塔贡）**：反望远结构，为 SLR 广角设计（法兰距长需要反望远让出反光板空间）
- **Biogon（标刚）**：对称广角，几乎零畸变，但只适用于短法兰距（旁轴机）

**无反时代的新趋势**：
- 短法兰距让广角镜头不再需要反望远结构 → 体积大幅缩小
- 但传感器斜入射光线导致边缘色偏和暗角 → 需要数字矫正

### 8.4 特殊镜片

| 镜片类型 | 功能 |
|----------|------|
| **ED / UD / SLD（超低色散）** | 减少色差（紫边） |
| **萤石（Fluorite）** | 最强的色差矫正，极贵，物理脆弱 |
| **非球面（ASPH / ASP）** | 矫正球差和畸变，减少镜片数量 |
| **DOE / PF（衍射光学）** | 大幅缩短长焦镜头体积（佳能 DO、尼康 PF） |
| **BR 镜片（蓝色光谱折射）** | 佳能专利用于彻底消除紫边 |
| **APO（复消色差）** | 三种波长的光汇聚到同一平面，几乎无色差 |

## 九、光学像差（Aberrations）

### 9.1 色差（Chromatic Aberration）

**纵向色差（LoCA / Axial CA）**
- 焦平面前后出现不同颜色的弥散斑（前紫后绿或前绿后紫）
- 大光圈逆光明显，收光圈可改善但无法根除
- 后期难以完全矫正

**横向色差（Lateral CA）**
- 画面边缘高反差边缘出现紫边/绿边
- 后期一键矫正（各软件自动校正功能）
- 不影响中心锐度

### 9.2 球差（Spherical Aberration）

- 镜片球面边缘的光线焦点与中心光线不一致
- 导致全开光圈时画面"柔"（但过渡反而柔和）
- 非球面镜片 + 收光圈 → 矫正

### 9.3 畸变（Distortion）

| 类型 | 表现 | 常见 |
|------|------|------|
| 桶形（Barrel） | 画面中心向外凸出 | 广角变焦 |
| 枕形（Pincushion） | 画面中心向内凹陷 | 长焦变焦 |
| 胡子形（Mustache） | 复合畸变，先桶后枕 | 部分广角头 |

后期可矫正，但会损失画面边缘分辨率。

### 9.4 慧差（Coma）

- 点光源在画面边缘拖出彗星状尾巴
- 影响星空摄影（边角星点变形）
- 大光圈广角最常见

### 9.5 像散（Astigmatism）

- 径切方向和切向方向的焦点不一致
- 边缘画质下降明显，无法通过收光圈完全消除

### 9.6 暗角（Vignetting）

- **光学暗角**：边缘光线被镜筒遮挡（余弦四次方定律）
- **机械暗角**：遮光罩太深、滤镜叠加
- 收光圈 2-3 档可大幅改善
- 后期可矫正，但代价是边缘信噪比下降

### 9.7 眩光与鬼影（Flare & Ghosting）

- **眩光（Veiling Flare）**：逆光时整体画面发灰、反差降低
- **鬼影（Ghost）**：光源在画面中出现彩色光斑排列（形状 = 光圈孔形）
- 镀膜越好，抗眩光越强
- 遮光罩可减少非画面光源的眩光

## 十二、感光度与噪点

### 12.1 噪点类型

| 类型 | 成因 | 特点 |
|------|------|------|
| **光子散粒噪声（Shot Noise）** | 光的量子涨落 | 无法消除，光越少越明显，是噪点根本来源 |
| **读取噪声（Read Noise）** | 电路读出信号时的电子噪声 | 好在近年传感器已很低 |
| **热噪声 / 暗电流** | 长曝光传感器发热 | 长曝光降噪可减（但翻倍时间） |
| **固定模式噪声** | 传感器像素间差异 | 现代传感器基本靠片上电路解决 |
| **彩色噪点** | 拜耳阵列插值 + 信号放大 | 高 ISO 典型表现 |

### 12.2 降噪策略

- 宁可保留颗粒感也不要把人磨成蜡像
- AI 降噪（DxO DeepPRIME、Lightroom AI NR、Topaz）效果碾压传统算法
- 彩色噪点比亮度噪点更影响观感，优先去除

## 十三、色彩与白平衡

### 13.1 色温参考

| 光源 | 色温 (K) |
|------|----------|
| 烛光 | 1800 ~ 2000K |
| 日出/日落 | 2500 ~ 3500K |
| 白炽灯（钨丝灯） | 2800 ~ 3200K |
| 暖白荧光 | 3500 ~ 4000K |
| 日光荧光 | 4000 ~ 4500K |
| 正午阳光 | 5200 ~ 5500K |
| 阴天 | 6000 ~ 7000K |
| 阴影/蓝天 | 7500 ~ 9000K |
| 灰卡 / 18% 灰 | 任意（以此为基准定义白平衡） |

### 13.2 白平衡模式

| 模式 | 行为 |
|------|------|
| **AWB** | 相机自动判断，偏暖场景容易被"校正"冷掉 |
| **AWB-W（白色优先）** | 把白色还原为纯白，保留原场景白 |
| **AWB-A（氛围优先）** | 保留暖调氛围，不完全校正 |
| **K 值手动** | 直接输入色温数字 |
| **自定义（Custom WB）** | 拍摄灰卡/白纸让相机校准 |

### 13.3 色彩空间

| 色彩空间 | 色域大小 | 用途 |
|----------|----------|------|
| **sRGB** | 最小（约 35% 可见光谱） | 网络分享、绝大多数屏幕 |
| **Adobe RGB** | 中（约 50%） | 印刷输出，青色和绿色范围更大 |
| **DCI-P3** | 中偏大 | 影视标准，苹果设备默认 |
| **ProPhoto RGB** | 极大 | 后期处理中间空间 |
| **Rec.2020** | 最大 | HDR 视频标准 |

**结论**：如果没有专业印刷需求，全程 sRGB 最省心。如果要打印，拍 Raw + Adobe RGB 备用。

### 13.4 照片风格 / 色彩模式

机内 JPEG 色彩配置（各品牌名称不同）：
- **标准（Standard）**：默认均衡
- **鲜艳（Vivid）**：高饱和高反差
- **中性（Neutral）**：低饱和低反差，最大化后期空间
- **人像（Portrait）**：保留肤色自然
- **风景（Landscape）**：增强蓝绿
- **单色（Monochrome）**：黑白
- **Flat / Log**（视频机）：最大动态范围的 Gamma 曲线，必须后期调色

## 十四、构图法则

### 14.1 核心法则

**三分法（Rule of Thirds）**：
- 画面横竖各三等分，交点即为视觉重心
- 地平线放在上或下 1/3 线
- 人像眼睛放在上交点

**引导线（Leading Lines）**：
- 道路、栏杆、河流、光线 → 引导视线指向主体
- 汇聚线扭曲成消失点 → 强烈纵深感

**框架构图（Frame within a Frame）**：
- 门窗、树枝、拱门圈出主体
- 增加层次感和窥视感

**对称构图**：
- 绝对的上下/左右对称
- 建筑、水面倒影
- 故意打破对称反而难受

**负空间（Negative Space）**：
- 主体很小，留白很大
- 极简、孤寂、高级感

### 14.2 进阶技巧

**前景-中景-远景层次**：
- 风光"一步一景"：前景石头/花 → 中景湖泊/树 → 远景雪山/天空
- 增加画面深度和立体感

**运动方向留白**：
- 人物/动物运动方向的前方多留空间
- 视线方向同理

**奇偶法则**：
- 画面中物体数量为奇数（3 个最优）比偶数更富动态和平衡感

**打破规则**：
- 中心构图可以强调力量和稳定
- 裁掉半边脸可以表达情绪
- 歪斜地平线制造不安感

### 14.3 常见画幅与视觉感

| 画幅比 | 视觉感 |
|--------|--------|
| 1:1（方形） | 平衡、安静，适合极简 |
| 3:2（135 原生） | 最经典，全面适配 |
| 4:3（M4/3、中画幅） | 更高更方，人像竖向更饱满 |
| 16:9 | 电影感、横向宽阔 |
| 65:24（宽幅 / XPAN） | 极宽视野，叙事感强 |
| 2.35:1（宽银幕） | 史诗电影感 |

## 十五、光

### 15.1 光质

| 类型 | 特征 | 典型光源 |
|------|------|----------|
| **硬光** | 清晰锐利的阴影边缘、高反差 | 正午阳光、裸灯、闪光直打 |
| **软光** | 柔和的阴影过渡、低反差 | 阴天、大柔光箱、窗边漫射光 |

决定光软硬的核心因素：**光源相对主体的大小**（不是距离）。太阳虽大但远 → 小光源 → 硬光。云层漫射 → 巨大光源 → 软光。

### 15.2 光位

| 光位 | 效果 |
|------|------|
| **顺光**（正面） | 平面化、色彩鲜艳、无阴影，但缺立体感 |
| **侧光**（90°） | 立体感最强，纹理突出 |
| **侧逆光** | 轮廓发光（Rim Light），分离主体与背景 |
| **逆光**（正后方） | 剪影、透射光（树叶、发丝发光）、氛围感强 |
| **顶光** | 正午太阳，眼窝和鼻子下方浓重阴影 |
| **底光** | 恐怖片专用，日常极少见 |

### 15.3 黄金时刻与蓝调时刻

- **黄金时刻（Golden Hour）**：日出后/日落前约 1 小时，光线温暖柔和，阴影拉长
- **蓝调时刻（Blue Hour）**：日出前/日落后约 20-40 分钟，天空深蓝，城市灯光亮起，对比极美
- **正午光**：光线最硬、色温最高，适合拍黑白或遮挡光线

### 15.4 室内光基础

**三点布光**（经典人像/产品/采访）：
1. **主光（Key Light）**：主要光源，决定整体光影基调
2. **辅光（Fill Light）**：提亮阴影，控制反差（反光板或第二盏灯）
3. **轮廓光（Rim / Hair Light）**：从背后或侧后方打亮轮廓，使主体从背景中分离

**光比**：主光与辅光的亮度比。2:1（自然）、4:1（立体）、8:1（戏剧化）。

## 十六、闪光灯

### 16.1 闪光灯分类

| 类型 | 特点 |
|------|------|
| **机顶闪光灯** | 热靴安装，便携，功率有限（GN 30-60） |
| **外拍灯** | 电池供电，功率中等（GN 60-100+） |
| **影室灯（Strobe）** | 交流供电，功率最大，可加各种附件 |
| **微距环闪** | 围绕镜头环形排布，无阴影微距专用 |
| **LED 常亮灯** | 所见即所得，视频友好，但功率远不如闪灯 |

### 16.2 核心概念

**GN（闪光指数）**：闪光功率。
- GN = 距离 × 光圈（ISO 100 时）
- 例：GN 60 → 6m 距离用 f/10、3m 距离用 f/20

**TTL（Through The Lens）**：相机通过镜头测光，自动计算闪光输出量。

**高速同步（HSS / Auto FP）**：突破闪光同步速度（1/200s），允许任意高速快门用闪光灯。代价是功率大幅下降。

**前帘同步 vs 后帘同步**：
- **前帘**：快门一开就闪 → 运动轨迹出现在主体前方（不自然）
- **后帘**：快门快关时才闪 → 运动轨迹跟在主体后方（自然）

**离机闪（Off-Camera Flash）**：
- 热靴线：稳定但受线缆限制
- 无线电引闪器：主流方案，多灯多组控制
- 光引闪（光脉冲 S1/S2）：老式方案，容易被误触发

### 16.3 常用附件

| 附件 | 效果 |
|------|------|
| **柔光箱**（Softbox） | 大面积柔光，窗形眼神光 |
| **反光伞（直打/透射）** | 柔化同时保留一定方向性 |
| **雷达罩（Beauty Dish）** | 介于硬软之间，人像经典 |
| **蜂巢（Grid）** | 限制光束范围，增加方向性 |
| **束光筒（Snoot）** | 极窄光束，点光源效果 |
| **色片（Gel）** | 改变光源颜色，模拟夕阳、创意色彩 |
| **反光板** | 金银白黑四面，补光/柔光/吸光 |

## 十七、滤镜

### 17.1 旋入式滤镜

| 滤镜 | 功能 |
|------|------|
| **UV / 保护镜** | 防刮防尘，对数码画质基本无贡献（会轻微降低反差） |
| **偏振镜（CPL）** | 消除非金属面反光（水面/玻璃/树叶）、压暗蓝天、提升饱和度。旋转调节，减光 1-1.5 档 |

| **柔光镜 / 黑柔（Black Mist）** | 高光扩散、降低锐度、胶片感。1/8 → 1/4 → 1/2 浓度递增 |
| **近摄镜（Close-up）** | 缩短最近对焦距离，给非微距镜头拍微距（画质折中方案） |

### 17.2 方片滤镜系统

适用于风光摄影，一片支架适配多种口径。品牌：Lee、NiSi、Kase、Haida。

## 二十、防抖（Image Stabilization）

| 类型 | 原理 | 效果 |
|------|------|------|
| **镜头防抖（OIS / VR / IS）** | 浮动镜组补偿抖动 | 长焦尤其有效 |
| **机身防抖（IBIS）** | 传感器位移补偿 | 所有镜头受益，包括老手动头 |
| **协同防抖（Sync IS）** | 机身 + 镜头联动 | 效果叠加，可达 6-8 档 |
| **电子防抖（EIS）** | 裁切 + 逐帧对齐 | 视频常用，损失视角 |

**注意事项**：
- 三脚架上应关闭防抖（可能引入反效果）
- 防抖防止的是你的手抖，不能防主体运动 → 快门速度仍需考虑
- 长焦防抖比广角重要得多

## 二十一、视频基础

### 21.1 分辨率

| 名称 | 像素 | 备注 |
|------|------|------|
| 1080p / Full HD | 1920×1080 | 基础交付 |
| 4K UHD | 3840×2160 | 当前主流 |
| 4K DCI | 4096×2160 | 电影标准 |
| 6K | ~6000 宽 | 裁切余量、超采样 |
| 8K UHD | 7680×4320 | 高端机身 |

### 21.2 帧率

| fps | 使用场景 |
|-----|----------|
| 24p | 电影感（180° 快门 = 1/48s） |
| 25p | 欧洲 PAL 广播标准 |
| 30p | 网络视频、日常 |
| 60p | 运动、流畅画面 |
| 120p+ | 慢动作（4×/5× 慢放） |

### 21.3 码率（Bitrate）

码率决定视频压缩程度。单位 Mbps（兆比特/秒）。
- 100Mbps → 足够 4K 30p 编辑
- 码率不足体现为运动画面糊成方块、暗部出现色块

**All-I vs IPB vs Long GOP**：
- **All-I（帧内）**：每帧独立编码，编辑友好但文件大
- **IPB / Long GOP**：只记录帧间差异，文件小但编辑吃配置

### 21.4 色彩采样（Chroma Subsampling）

| 采样 | 意义 |
|------|------|
| 4:4:4 | 每像素都有完整色彩信息，调色最佳 |
| 4:2:2 | 水平方向每两个像素共享色彩 |
| 4:2:0 | 水平和垂直各两个像素共享色彩（YouTube/蓝光标准） |

### 21.5 视频辅助功能

- **斑马纹**：显示画面中亮度过曝的区域（通常设 90-100%）
- **峰值对焦**：以彩色颗粒标出合焦边缘
- **示波器**：波形图、矢量图辅助判断曝光和色彩
- **LUT**：预览用色彩查找表，模拟后期风格
- **Log 伽马**：S-Log、C-Log、V-Log、N-Log，最大化动态范围
- **RAW 视频**：ProRes RAW、BRAW、N-RAW，后期自由度等同照片 Raw

### 21.6 快门角度 vs 快门速度

视频领域的 180° 规则：快门速度 = 1 / (2 × 帧率)。
- 24fps → 1/48s（180°）
- 60fps → 1/120s
- 角度越小 → 每一帧越清晰但运动越卡顿（《拯救大兵瑞恩》的 45° / 90° 快门的紧张感）

## 二十五、选购思路

### 优先投资顺序

1. **镜头 > 机身**：好镜头用十年，好机身三年过时
2. **定焦入门**：50mm f/1.8 最便宜的定焦画质远超最贵的变焦
3. **先拍够再升级**：器材焦虑是创造力的敌人
4. **二手优先**：镜头、三脚架、滤镜买二手非常划算
5. **买你能承受的最好的三脚架**：差的三脚架会让你不想用它，最后浪费两头钱

### 常见搭配思路

- **入门全画幅**：二手 A7III / Z6 / EOS R + 50mm f/1.8 + 24-70mm f/4
- **旅行轻便**：M4/3 机身 + 12-100mm f/4（等效 24-200mm）
- **人像**：全画幅 + 85mm f/1.4（或 f/1.8）
- **风光**：全画幅 + 16-35mm f/2.8 + 100-400mm + 稳固三脚架 + CPL + ND
- **街头纪实**：APS-C 或全画幅 + 35mm（或等效 35mm）+ 28mm
- **视频为主**：索尼 FX 系列 / A7S 系列 / 松下 S5 / 佳能 C 系列
